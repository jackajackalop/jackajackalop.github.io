This is my capstone project for my degree, a two semester semester long independent project I completed under the guidance and endless patience of Jim McCann. With the permission of the lovely and amazing Victo Ngai I designed and implemented a system of stylization shaders inspired by her artwork. This example scene is modeled after "Sweet Dreams" which can be found in the New Yorker Collection of her website.

This is made using C++, OpenGL, and the base code for 15-466 at CMU. My code can be found here.
